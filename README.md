# Node + TypeScript template


* Already configured tsconfig.json
* Uses super fast SWC TypeScript compiler

### Get Started

1. Clone that
```
git clone https://github.com/pheianox/node-ts-boil
cd node-ts-boil/
```

2. Install
```shell
pnpm install # or npm/yarn
```

3. Run 
```shell
pnpm run dev # or npm/yarn
```
